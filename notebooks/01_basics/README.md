## Main Takeaways So Far

- linear regression is the cleanest entry point for understanding forward pass, loss, and gradients
- backpropagation becomes easier to understand when you first see it in the one-neuron setting
- a single perceptron can solve linear boundaries but cannot solve XOR
- hidden layers create the non-linear feature transformations that solve XOR
- activation functions are not cosmetic choices; they affect trainability and initialization strategy
