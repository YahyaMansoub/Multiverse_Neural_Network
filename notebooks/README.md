# Notebooks

This folder now separates notebooks by learning track so the project can grow without becoming hard to navigate.

## Folder Map

- `01_basics/` foundational math and neural-network building blocks
- `02_ai_vision/` future computer vision notebooks
- `03_natural_language/` future NLP notebooks
- `04_deep_learning/` future deep-learning architectures and training studies
- `99_archive/` preserved alternate or older notebook versions

## Current Insights Captured

The notebooks already establish a strong first-principles sequence:

- linear regression shows the full gradient story in the simplest setting
- perceptrons introduce activations and binary cross-entropy
- XOR shows why hidden layers are necessary
- activation choice affects gradient flow, expressiveness, and initialization

That sequence is now reflected in the new Python library under `python/`.
